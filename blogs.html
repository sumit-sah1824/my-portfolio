üß† Gradient Descent and Convergence

Consider a quadratic loss function:

\[ f(x) = \frac{1}{2}x^2 \]

The gradient is \( \nabla f(x) = x \), so the update becomes:

\[ x_{t+1} = x_t - \eta x_t = (1 - \eta)x_t \]

This results in exponential decay when \( 0 < \eta < 1 \).

üîÅ Projection in Online Learning

In constrained optimization, we use projection:

\[ x_{t+1} = \Pi_{\mathcal{X}}(x_t - \eta \nabla f_t(x_t)) \]

Where \( \Pi_{\mathcal{X}} \) projects back into feasible set \( \mathcal{X} \).
